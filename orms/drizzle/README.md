# Gold Lapel тАФ Drizzle

Drop-in Drizzle integration тАФ import `drizzle` from `goldlapel-drizzle` instead of `drizzle-orm`.

## Setup

1. Start Postgres: `docker compose up -d`
2. Install deps: `npm install`
3. Run the app: `node app.js`

## What to look for

GL starts automatically when Drizzle connects. Check the dashboard at http://localhost:7933.
